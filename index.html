4. What Is HTTP?

HTTP means HyperText Transfer Protocol. It is the language browsers and programs use to communicate with websites and APIs.

When Java asks the weather API for data, it sends an HTTP request.

HttpRequest request =
    HttpRequest.newBuilder()
        .uri(URI.create(url))
        .GET()
        .build();

5. What Is JSON?

JSON means JavaScript Object Notation. It is a lightweight format used to store and transfer data.

JSON looks like this:

{
  "temperature": 28.7,
  "humidity": 32,
  "wind_speed": 6.0,
  "condition": "Clear sky"
}

Think of JSON like a labeled box. Each label has a value.

Key

"temperature"

Value

28.7

String Value

"Clear sky"

6. What Is Parsing?

Parsing means reading raw data and extracting the useful parts.

Raw JSON:

{
  "current": {
    "temperature_2m": 83.6,
    "relative_humidity_2m": 32,
    "wind_speed_10m": 3.7
  }
}

Java parsing with Jackson:

JsonNode root = mapper.readTree(json);

double temperature =
    root.get("current")
        .get("temperature_2m")
        .asDouble();
Result: Temperature = 83.6

Why do we need latitude and longitude?

Weather APIs usually need exact coordinates, not just a city name. So the app first converts the city name into coordinates.
